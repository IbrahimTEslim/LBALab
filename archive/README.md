# Archive - Legacy Files

This directory contains the original development files and legacy implementations.

## Contents

### Original Development Files
- `lba.py`, `r_lba.py`, `w_lba.py` - Early LBA operation prototypes
- `res.py`, `res2.py`, `lba_resd.py` - Residency detection experiments
- `f_2_lba.py`, `pf_lba.py` - File-to-LBA mapping prototypes
- `mft_fixed.py` - MFT analysis development version
- `ntfs_lba_tool.py` - Early unified tool attempt

### C++ Implementations
- `w_lba.cpp`, `w2_lba.cpp` - C++ LBA writing implementations

### Development Tools
- `deep_write.py` - Deep disk writing experiments
- `lba_direct.py` - Direct LBA access testing
- `lba_safe.py` - Safe LBA operations prototype

## Purpose

These files are preserved for:
- **Historical Reference**: Track development evolution
- **Code Archaeology**: Understand design decisions
- **Learning**: See different implementation approaches
- **Recovery**: Access specific functionality if needed

## Status

These files are **deprecated** and should not be used in production. All functionality has been integrated into the main toolkit in the `tools/` directory.
